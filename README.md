# 郑世怀个人作品集

机械设计工程师个人作品集基础版，使用 React + Vite 开发。

## 本地运行

```powershell
npm install
npm run dev
```

生产构建：

```powershell
npm run build
```

## 内容维护

- 页面内容与项目数据：`src/App.jsx`
- 视觉样式与响应式：`src/styles.css`
- 图片、视频和简历：`public/assets/`
- 视觉规范：`.kun-design/DESIGN_SYSTEM.md`

项目卡片当前使用已本地化的授权工业影像作为概念视觉。拿到真实 CAD 渲染、装配图或样机照片后，直接替换同名图片即可。
