# 作品集设计规范

- 方向：暗色、克制、精密工业感；强调机械图纸、坐标、参数与材料质感。
- 主色：深黑 `#090b0d`，面板 `#101316`，正文 `#f2f3ef`，辅助文字 `#9da3a6`。
- 强调色：安全黄 `#d9ff43`，只用于关键动作、编号和状态。
- 字体：正文使用系统黑体，首页大标题使用窄体工程展示字，数字与参数使用等宽字体。
- 圆角：卡片 4px，按钮 2px，不使用大圆角胶囊。
- 版心：最大 1700px，左右留白随视口自适应。
- 动效：使用 GSAP + ScrollTrigger。首屏采用视频缩放、遮罩揭开、导航滑入及标题压缩归位的完整开场；模块标题先以英文大字横向拉伸进入，再以裁切揭开中文标题；卡片按序错峰进入，项目图使用裁切揭示和低频滚动视差。整体节奏偏慢，使用 `expo`、`power3`、`power4` 缓动，不使用弹跳或弹性效果；磁流体背景保持持续流动，不因系统动态偏好降级。
- 导航：首屏使用透明覆盖状态，离开首屏后固定顶部并切换为深色磨砂玻璃状态。
- 内容背景：头图以下至联系页之前使用低对比度黑白磁流体轮廓，作为粘性环境层；不得遮挡正文和操作控件。
- 经历卡片：使用安全黄、冷白与金属灰的边缘扫光；卡片圆角保持 4px，鼠标靠近边缘时增强轮廓和外发光。
